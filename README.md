# WinPrivChecker
Auto Windows Privilege Escalation Checker

## Install
```sh
git clone https://github.com/RajChowdhury240/WinPrivChecker.git
cd WinPrivChecker
sudo chmod +x wincheck
sudo cp wincheck /usr/local/bin
```
### Usage
```sh
wincheck  -u '<username>' -d <domain> -p '<Password>' -t <IP>
```
